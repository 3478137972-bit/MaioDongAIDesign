## 1. 功能介绍

### 1.1 产品概述

AI 绘图和无限画布功能是 AI 设计工具平台的核心能力之一。该功能结合了 DeepSeek 智能提示词生成和 KIEAI 图像生成技术，为用户提供流畅的 AI 创作体验。

### 1.2 核心功能

| 功能模块 | 功能描述 |
|---------|---------|
| **智能对话** | 通过自然语言描述需求，AI 自动补充细节并生成专业提示词 |
| **多模型支持** | 支持三种图像生成模型：Nano Banana Pro、Nano Banana、Seedream 4.5 |
| **尺寸灵活** | 支持多种宽高比（1:1、2:3、3:2、16:9 等）和分辨率（2K、4K） |
| **无限画布** | 支持图片的无级缩放和自由拖拽，方便查看细节 |
| **多轮对话** | 支持多轮交互，逐步完善创作需求 |
| **历史记录** | 自动保存创作历史，支持回顾和复用 |

## 2. 使用教程

### 2.1 快速开始

#### 步骤 1：选择模型

在界面左侧的模型选择器中，选择适合您需求的图像生成模型：

```
┌────────────────────────────────────┐
│  模型选择                          │
│  ┌──────────────────────────────┐  │
│  │ Nano banana pro          ▼  │  │
│  └──────────────────────────────┘  │
│  ┌──────────────────────────────┐  │
│  │ 1:1                       ▼  │  │
│  └──────────────────────────────┘  │
│  ┌──────────────────────────────┐  │
│  │ 2K                        ▼  │  │
│  └──────────────────────────────┘  │
└────────────────────────────────────┘
```

> 💡 **提示**：首次使用时建议使用默认的 Nano Banana Pro 模型，它支持最全面的尺寸和分辨率选项。

#### 步骤 2：输入创作需求

在底部输入框中描述您想要生成的图像内容：

```
┌─────────────────────────────────────────────────────────────┐
│ [请描述您想要生成的图像...]                        [发送]   │
└─────────────────────────────────────────────────────────────┘
```

**示例输入**：
- "一只可爱的橘猫坐在窗台上，窗外是黄昏的夕阳"
- "未来科技感的城市夜景，霓虹灯闪烁"
- "水彩风格的山水画，远山云雾缭绕"

#### 步骤 3：AI 补充细节

AI 可能会向您提问以获取更多细节。请根据提示回答问题，帮助 AI 更准确地理解您的需求。

```
┌─────────────────────────────────────────────────────────────┐
│ 🤖 请补充以下信息：                                         │
│                                                             │
│ ▸ 图片风格？[写实 / 卡通 / 抽象 / 水彩]                    │
│ ▸ 颜色基调？[暖色 / 冷色 / 中性]                            │
│ ▸ 画面比例？[横版 / 竖版 / 正方形]                          │
└─────────────────────────────────────────────────────────────┘
```

#### 步骤 4：查看生成结果

回答完问题后，AI 将开始生成图像。生成完成后，结果将显示在右侧的无限画布区域。

```
┌─────────────────────────────────────────────────────────────┐
│  生成完成！                                                 │
│  ┌─────────────────────────────────────────────────────┐   │
│  │                                                     │   │
│  │              [生成的图像预览]                       │   │
│  │                                                     │   │
│  │                                                     │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  📝 提示词：一只可爱的橘猫坐在窗台上，窗外是黄昏...       │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 无限画布操作指南

无限画布是本功能的核心特性之一，支持以下操作：

#### 缩放操作

| 操作 | 方法 |
|------|------|
| **放大** | 鼠标滚轮向前滚动 / 点击 + 号按钮 |
| **缩小** | 鼠标滚轮向后滚动 / 点击 - 号按钮 |
| **适应窗口** | 双击画布 / 点击适应窗口按钮 |

```
缩放范围：0.5x - 3x（推荐）
```

> ⚠️ **注意**：过小的缩放（<0.5x）可能导致图片难以辨认，过大的缩放（>3x）可能影响性能。

#### 拖拽操作

| 操作 | 方法 |
|------|------|
| **移动画布** | 按住鼠标左键拖拽 |
| **重置位置** | 点击重置按钮或双击 |

#### 图片切换

如果有多个生成结果，可以使用下方的导航按钮进行切换：

```
[<]  [1/3]  [>]
```

### 2.3 图片下载

生成完成后，您可以点击下载按钮将图片保存到本地：

1. 找到生成结果下方的 **下载** 按钮
2. 点击按钮，图片将自动保存为 PNG 格式
3. 如果下载失败，请检查网络连接后重试

## 4. 常见问题

### 4.1 功能使用问题

#### Q1: 生成图像失败怎么办？

**可能原因**：
- 网络连接不稳定
- API 密钥配置错误
- 服务器负载过高

**解决方法**：
1. 检查网络连接是否正常
2. 刷新页面后重试
3. 稍后再次尝试（避开高峰期）

#### Q2: 生成的图像不符合预期？

**解决方法**：
1. 使用多轮对话补充更多细节
2. 尝试调整尺寸或模型
3. 重新描述需求，使用更具体的描述

#### Q3: 图片缩放没有反应？

**可能原因**：
- 缩放范围超出限制
- 浏览器兼容性问题

**解决方法**：
1. 确认缩放在 0.5x - 3x 范围内
2. 尝试刷新页面
3. 使用 Chrome 或 Edge 浏览器

#### Q4: 下载的图片是空白的？

**可能原因**：
- 图片还在加载中
- 网络中断导致图片未完全加载

**解决方法**：
1. 等待图片完全加载后再下载
2. 检查网络连接
3. 刷新页面重新生成

### 4.2 配置问题

#### Q5: 提示需要配置 API 密钥？

**说明**：首次使用需要配置环境变量。请参考部署指南中的环境配置部分。

#### Q6: 如何切换不同的图像模型？

**操作方法**：
1. 在左侧面板找到「模型选择」下拉框
2. 点击选择所需的模型
3. 系统会自动加载该模型支持的尺寸选项

### 4.3 性能问题

#### Q7: 页面加载缓慢？

**可能原因**：
- 网络带宽不足
- 同时打开多个标签页
- 浏览器缓存过多

**解决方法**：
1. 关闭不需要的标签页
2. 清除浏览器缓存
3. 尝试使用有线网络

#### Q8: 生成时间过长？

**说明**：图像生成需要一定时间，具体取决于：
- 图像分辨率（4K 比 2K 慢）
- 服务器负载
- 网络状况

**建议**：耐心等待或选择较低的分辨率

### 4.4 错误提示说明

| 错误信息 | 含义 | 解决方法 |
|---------|------|---------|
| `网络错误，请检查连接` | 网络不稳定 | 检查网络后重试 |
| `请求超时` | 等待时间过长 | 刷新页面重试 |
| `API 密钥无效` | 配置错误 | 检查环境变量配置 |
| `服务暂时不可用` | 服务器繁忙 | 稍后重试 |
