## 🐛 作品集灯箱 Bug 修复

**时间**: 2026-03-20 19:35 GMT+8

**问题**: 图片无法点击放大，灯箱不显示

**根本原因**: 
- `classList.remove('hidden')` 被 CSS 覆盖
- 灯箱层级不够高

**修复方案**:
1. 改用 `style.display = 'flex/none'` 直接控制
2. 提升灯箱层级至 `z-[9999]`
3. 添加初始 `style="display: none;"`
