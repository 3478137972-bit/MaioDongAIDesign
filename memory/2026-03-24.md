## 技术要点

### 图片路径
- 微信小程序必须使用**相对路径**加载本地图片
- 正确格式：`../../static/images/...`
- 错误格式：`/static/images/...` (会导致 500 错误)

### 设计规范
- 品牌色：#F97316 (橙色渐变)
- 圆角：24rpx
- 卡片阴影：0 8rpx 32rpx rgba(0,0,0,0.06)
- 移除所有 emoji，使用 CSS 图标类

### Git 推送
- 使用 HTTPS + token 方式
- SSH 配置有问题（服务器未配置）
- 命令：`git push origin main`
