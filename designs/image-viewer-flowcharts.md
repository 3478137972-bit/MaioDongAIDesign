## 👆 拖拽平移详细流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          拖拽平移详细流程                                    │
└─────────────────────────────────────────────────────────────────────────────┘

              ┌─────────────────────────┐
              │  拖拽开始 (touchstart)   │
              │  条件：缩放比例 > 100%   │
              └───────────┬─────────────┘
                          │
                          ▼
              ┌─────────────────────────┐
              │  记录初始触摸位置        │
              │  (startX, startY)        │
              │  记录当前图片偏移        │
              │  (currentX, currentY)    │
              └───────────┬─────────────┘
                          │
                          ▼
              ┌─────────────────────────┐
              │  拖拽中 (touchmove)      │
              │  计算位移量：            │
              │  deltaX = curX - startX  │
              │  deltaY = curY - startY  │
              └───────────┬─────────────┘
                          │
                          ▼
              ┌─────────────────────────┐
              │  计算新位置：            │
              │  newX = currentX + deltaX│
              │  newY = currentY + deltaY│
              └───────────┬─────────────┘
                          │
                          ▼
              ┌─────────────────────────┐
              │  边界检测                │
              │  ┌───────────────────┐  │
              │  │ 图片边界 =        │  │
              │  │ (图片尺寸 - 视口) │  │
              │  │ * 缩放比例        │  │
              │  └───────────────────┘  │
              └───────────┬─────────────┘
                          │
                    ┌─────┴─────┐
                    │           │
                    ▼           ▼
              未超出边界    超出边界
                    │           │
                    │           ▼
                    │    ┌──────────────┐
                    │    │ 计算弹性效果  │
                    │    │ overshoot =  │
                    │    │ distance *   │
                    │    │ 0.15         │
                    │    └──────┬───────┘
                    │           │
                    │           ▼
                    │    ┌──────────────┐
                    │    │ 显示视觉反馈  │
                    │    │ - 半透明阴影  │
                    │    │ - 透明度变化  │
                    │    └──────┬───────┘
                    │           │
                    │           ▼
                    │    ┌──────────────┐
                    │    │ 限制位置在    │
                    │    │ 边界 +        │
                    │    │ overshoot     │
                    │    └──────┬───────┘
                    │           │
                    └─────┬─────┘
                          │
                          ▼
              ┌─────────────────────────┐
              │  应用变换：              │
              │  transform:              │
              │  translate(newX, newY)   │
              └───────────┬─────────────┘
                          │
                          ▼
```
