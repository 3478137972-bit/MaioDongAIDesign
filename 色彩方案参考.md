## 💡 使用建议

### 60-30-10 法则
- **60% 主色**: 背景、大面积区域（白色/浅灰）
- **30% 辅助色**: 卡片、导航、按钮（品牌蓝）
- **10% 强调色**: CTA 按钮、重要提示（青色/紫色）

### 对比度要求
- 文字与背景对比度至少 4.5:1 (WCAG AA)
- 重要 UI 元素对比度至少 3:1
- 使用工具检查：https://webaim.org/resources/contrastchecker/

### 暗色模式适配
```css
/* 暗色模式配色 */
--bg-dark: #0F172A;
--surface-dark: #1E293B;
--text-dark-primary: #F1F5F9;
--text-dark-secondary: #94A3B8;
```
